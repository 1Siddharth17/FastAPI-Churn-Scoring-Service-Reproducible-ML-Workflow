# Customer Churn Prediction API

## Part 4: FastAPI Churn Scoring Service & Reproducible ML Workflow

---

# Project Overview

This project deploys a machine learning churn prediction model as a FastAPI service.

The API allows CRM systems, customer success teams, and retention teams to obtain churn-risk predictions for individual customers or groups of customers.

The deployed model was trained using customer behavioral, transactional, and support-related features and predicts whether a customer is likely to churn within the next 60 days.

---

# Business Objective

The purpose of this API is to:

* Identify customers at risk of churn
* Support proactive retention campaigns
* Provide churn probabilities for decision-making
* Enable real-time customer scoring
* Integrate with CRM or marketing systems

```json
  "api_version": "1.0.0"

```

---

## POST /predict

Predict churn risk for a single customer.

### Sample Request

```json
{
  "customer_id": "C102",
  "recency_days": 90,
  "frequency": 2,
  "monetary_value": 500,
  "avg_order_value": 250,
  "support_ticket_count": 4,
  "unresolved_tickets": 2,
  "intervention_count": 1,
  "tenure_days": 365,
  "return_rate": 0.30,
  "plan_type_encoded": 1
}
```

### Sample Response

```json
{
  "customer_id": "C102",
  "churn_probability": 0.81,
  "predicted_class": 1,
  "risk_level": "High",
  "risk_explanation": "Low activity and multiple support tickets indicate elevated churn risk.",
  "confidence": 0.81
}
```

---
